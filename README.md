# Module_MultiAgent--17


## Project Description
This project is an AI-powered SQL Agent built using **LangChain**, **SQLite**, and **Streamlit**.  
It can query information about **Hospitals**, **Restaurants**, and **Institutions** in Bangladesh.  
The agent interacts with multiple databases and answers natural language queries.

## Features
- Query multiple domains (Hospitals, Restaurants, Institutions)
- Auto-inspects database schemas for correct queries
- Streamlit-based interactive web app
- Handles multiple tables in SQLite databases
